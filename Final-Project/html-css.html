Introduction to HTML and CSS

- HTML and CSS Logos. Credits: Lambdatest.com

By Hamza Elafifi

Written October 19, 2021

HTML

Hypertext Markup Language, or HTML for short, is the coding language used to display content on websites such as text, images, videos, etc, by using HTML tags. HTML tags are specific pieces of HTML that are used to display different types of content. For example, a paragraph tag would use a different tag than an image tag "p and img". Whatever is written inside the HTML tag is known as the elements. This could be the text inside the paragraph or the content inside the div. For images, links, and videos, HTML attributes are needed. HTML attributes provide additional information about the elements. The href attribute is used to place the link for the link tag and the src attribute is used to locate the image for the image tag. Inline elements are used to occupy the space needed within the main element while block elements are any elements that start new lines and occupy the space of the page or the container.

- HTML elements example. Credits: Hamza Elafifi

CSS

Cascading Style Sheets, or CSS for short, is a coding language that is used to style the content on websites such as changing colours, content sizes, creating animations, etc, and we use CSS rules for such. CSS rules are used to group one or more styling properties which are to be applied to one or possibly more HTML elements on the webpage. This could be grouping the styling for the sizing, color and/or animation of an element or many elements. The CSS selector is used to target the HTML element that we want to style which can either be selected based on the element type or if it has an ID that is used to style it separately from other similar elements. CSS values are used to select the desired outcome of the styling. This includes choosing the color you want or the exact dimensions of the desired HTML element.
